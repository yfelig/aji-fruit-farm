# Your Website — Owner Guide

Welcome Aji! This is your website. You own it. Here's how to keep it updated.

---

## To update your WhatsApp number

Open `index.html` in a text editor. Use Find & Replace (Cmd+H on Mac) to find:
```
94XXXXXXXXX
```
Replace with your actual number in international format — no `+`, no spaces.  
Example: `94771234567`

---

## To update photos

1. Go to [squoosh.app](https://squoosh.app) and compress your photo to under 500KB
2. Save it to the `photos/` folder with the exact same filename as the one you're replacing
3. The site updates automatically when pushed to GitHub

**Photo files to replace:**
| File | What it shows |
|------|--------------|
| `photos/hero.jpg` | The big full-screen opening photo |
| `photos/cottage-exterior.jpg` | Cottage from outside (The Place section) |
| `photos/aji.jpg` | Your portrait (Meet Aji section) |
| `photos/gallery-lake.jpg` | Lake view |
| `photos/gallery-bathroom.jpg` | The open-air bathroom |
| `photos/gallery-interior.jpg` | Inside the cottage |
| `photos/gallery-mangos.jpg` | The mango orchard |
| `photos/gallery-veranda.jpg` | Veranda with lake view |
| `photos/gallery-kayak.jpg` | Kayaking on the lake |

## To update text

Open `index.html`, use Find & Replace to change any text you see on the page.

---

## To update the booking message

Find the lines starting with `href="https://wa.me/94...` and edit the text after `?text=`.  
Spaces = `%20`, apostrophes = `%27`.
